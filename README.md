# Dawn of the Defender

A fast-paced 2D top-down zombie shooter built entirely in JavaScript using p5.js without external game engines or frameworks.

## Play the Game

🎮 itch.io:  
https://shayanbhatti.itch.io/dawn-of-the-defender
## Overview

Dawn of the Defender is a survival-based action game set in a post-apocalyptic world overrun by zombies. Players fight through waves of enemies while managing ammunition, dodging attacks, and battling a final boss encounter.

This project was developed from scratch using p5.js, with custom implementations for:
- enemy AI
- collision detection
- animation systems
- projectile physics
- boss mechanics
- game state management
- camera movement
- audio systems

## Features

- Combat with zombies spawned over the map
- Dynamic boss battle with multiple attack patterns
- Ammo and reload mechanics
- Multiple playable character styles
- Health and damage systems
- Audio effects and background music
- Full-screen game world with camera tracking
- Animated sprites and custom UI
- Randomized enemy movement speeds and attack behavior

## Technologies Used

- JavaScript
- p5.js
- HTML/CSS

## Technical Highlights

- Built entirely without external game development frameworks
- Implemented object-oriented game entities using JavaScript prototypes
- Developed custom collision systems using bounding box detection
- Used trigonometry (`atan2`, sine, cosine) for:
  - enemy tracking
  - projectile movement
  - rotation systems
- Designed modular enemy, bullet, and boss systems
- Created sprite animation cycles and camera-relative rendering
